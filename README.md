# Purpose
This Meal Plan Project is intended first and foremost to demonstrate coding techniques using nodejs and javasrcipt.

The Models include:

- Recipe
  - Title
  - Number of Ingredients
  - Calories
<br />
<br />
- Meal 
  - Date
  - Type  (Breakfast, Lunch Dinner)
  - Recipe Index Array
<br />
<br />
- Reviews
  - Recipe Index
  - Name of Reviewer
  - Number of Stars 1 to 5
